☁️ Sincronización GitHub Gist — Guía
La sincronización con GitHub Gist permite que tus datos se guarden en la nube y estén disponibles en cualquier dispositivo. Es gratuito y solo necesitas una cuenta de GitHub.
Paso 1 — Crear cuenta GitHub Ve a github.com y crea una cuenta gratuita si no tienes una.
Paso 2 — Crear un Token de acceso 1. Pulsa tu foto de perfil (arriba a la derecha) → Settings
2. Baja hasta el final → Developer settings
3. Personal access tokens → Tokens (classic)
4. Generate new token (classic)
5. Ponle un nombre (ej: "CUADRANTE"), selecciona el scope gist y pulsa Generate token
6. Copia el token — solo se muestra una vez
Paso 3 — Crear un Gist 1. Ve a gist.github.com
2. Crea un nuevo Gist: pon cualquier nombre de archivo (ej: cuadrante.json), escribe cualquier cosa en el contenido y pulsa Create secret gist
3. Copia el ID del Gist — es la parte final de la URL: gist.github.com/tuusuario/abc123def456
Paso 4 — Configurar en la app Ve a Ajustes → Sincronización GitHub Gist e introduce:
• Token: el que copiaste en el paso 2
• Gist ID: el que copiaste en el paso 3
Pulsa Guardar y sincronizar. A partir de ahora tus datos se sincronizan automáticamente.
Compartir con usuarios Cada usuario necesita su propia cuenta GitHub y su propio Gist. La sincronización es individual — cada uno guarda sus propios datos en la nube.
